[ 인텔리제이 깃허브 연동 ]
    1. 인텔리제이 상단메뉴 -> [파일] -> [설정]
    2. 사이드메뉴 [버전관리] ->[github]
    3. [계정추가] --> github통해 --> 웹 깃허브 로그인후 인증

[ 인텔리제이 깃허브 커밋계정 정보 ]
    1. 인텔리제이 왼쪽 하단 메뉴 [ 터미널 : alt + f12 ]
    2. 터미널 왼쪽 상단 [v] 아이콘클릭 --> [git bash]
        git config --global user.name 본인아이디
        git config --global user.email 본인깃메일

[ 인텔리제이 에서 깃허브에 프로젝트 게시하기 ]
    1. 인텔리제잉 상단메뉴 -> [ VCS ]
    2. [ Github 프로젝트 공유 ]
        - 저장소 이름 : 프로젝트명(*아무거나) !!! [체크안함] 비공개
        - 원격 : (기본값)
